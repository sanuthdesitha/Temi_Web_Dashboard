KNOWN LIMITATIONS
================================================================================

Current Version (1.0.0):
  • Single Flask development server (can be upgraded to production server)
  • SQLite database (suitable for small-medium deployments, can be upgraded)
  • No built-in LDAP/Active Directory (local user accounts only)
  • Requires manual MQTT broker configuration
  • Dashboard refresh time ~1 second (WebSocket based)

Recommended Upgrades:
  • Production WSGI server (Gunicorn, uWSGI) for load balancing
  • PostgreSQL/MySQL for large-scale deployments (100+ robots)
  • Enterprise authentication integration (LDAP, OAuth)
  • Horizontal scaling with load balancer
  • Kubernetes deployment for high availability
