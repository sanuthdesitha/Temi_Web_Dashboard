## Short Description (2-3 sentences)

The Temi Robot Control WebApp is a comprehensive management and monitoring system for autonomous mobile robots. It provides a unified web interface to control multiple Temi robots, execute complex autonomous patrols with violation detection, and generate compliance reports‚Äîall integrated with HiveMQ Cloud MQTT for remote access and real-time monitoring.
